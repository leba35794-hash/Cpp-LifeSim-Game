# Cpp-LifeSim-Game

一个简单的C++人生模拟游戏

## 当前版本

Alpha 0.15

## 游戏功能

- 打工赚钱
- 背包系统
- 彩票系统
- 天数系统
- 目标系统

## 怎么编译

需要C++17编译器，编译命令：

```bash
g++ Main.cpp events.cpp -o game
```

运行命令：

```bash
./game
```
